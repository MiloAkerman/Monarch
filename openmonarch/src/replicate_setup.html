In order to use the image description feature, you need to be connected to the servers that analyze the images. To connect to the servers, please follow the guide below:

  1. Follow this link to the GitHub login screen
  2. Log in with your GitHub account. If you do not have one, click on the link reading "Create an Account"
  3. Back on the Replicate website, look for a long string of characters under "API Key", and copy it
  4. Paste the string you copied in the textbox below, and click "Submit"
